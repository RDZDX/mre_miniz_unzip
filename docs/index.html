Mre_miniz_tools

App demonstrate how to zip/unzip files for MRE platform mobile phone (including Nokia S30+). Zip file size limited by phone memory. Tested on Nokia 225 with display resolution 240x320. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application files - "miniz_zip.vxp"; "miniz_unzip.vxp"; "miniz_ungz.vxp";
