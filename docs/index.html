Mre_miniz_unzip

App demonstrate how to unzip files for MRE platform mobile phone (including Nokia S30+). RIGHT_SOFTKEY - exit app. Tested on Nokia 225 with display resolution 240x320. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application file - "mre_miniz_unzip.vxp"; mre_miniz_unzip.vxp - works with blocks of files (not in memory).
